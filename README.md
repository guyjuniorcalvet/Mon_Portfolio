# My Interactive Portfolio

This portfolio was built with Python and Streamlit to showcase my skills in a live, interactive web application.

## How to Run Locally

1.  **Navigate to the project directory:**
    ```bash
    cd "/Users/guyjuniorcalvet/Library/CloudStorage/OneDrive-UniversitéduQuébecàChicoutimi/Documents/Projet/PORTFOLIO/"
    ```

2.  **Install the dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the Streamlit app:**
    ```bash
    streamlit run portfolio_app.py
    ```
